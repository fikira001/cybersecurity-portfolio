# Conduct a Security Audit

## Overview

This portfolio activity demonstrates the process of conducting an internal security audit for a fictional organization, Botium Toys, using the NIST Cybersecurity Framework (CSF).

## Objectives

- Review the organization's security posture
- Identify existing security controls
- Evaluate compliance requirements
- Assess risks to critical assets
- Recommend security improvements

## Skills Demonstrated

- Security Auditing
- Risk Assessment
- Security Controls
- Compliance
- NIST Cybersecurity Framework
- Documentation

## Deliverables

- Security Audit Report
- Controls and Compliance Checklist

## Key Takeaways

- Learned how to evaluate an organization's security controls.
- Applied the NIST Cybersecurity Framework during an internal audit.
- Identified gaps affecting security and regulatory compliance.
- Documented findings using professional security audit practices.
