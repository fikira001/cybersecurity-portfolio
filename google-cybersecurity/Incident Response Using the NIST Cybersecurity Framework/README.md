# Incident Response Using the NIST Cybersecurity Framework

## Overview

This portfolio activity demonstrates the application of the NIST Cybersecurity Framework (CSF) to analyze and respond to a simulated cybersecurity incident involving a Denial-of-Service (DoS) attack.

## Scenario

A multimedia company experienced a DoS attack after an attacker exploited an improperly configured firewall, flooding the network with ICMP traffic. The incident disrupted internal network services for approximately two hours before being contained.

## Objectives

- Analyze the security incident
- Apply the NIST Cybersecurity Framework
- Develop recommendations to improve network security
- Document an incident response process
- Propose recovery and future mitigation strategies

## Skills Demonstrated

- Incident Response
- NIST Cybersecurity Framework
- Network Security
- Risk Analysis
- Security Documentation
- DoS Attack Analysis

## Framework Used

- **Identify**
- **Protect**
- **Detect**
- **Respond**
- **Recover**

## Deliverables

- Incident Report Analysis
- NIST CSF Response Plan

## Key Takeaways

- Applied the five functions of the NIST Cybersecurity Framework to a real-world scenario.
- Identified weaknesses in firewall configuration and network monitoring.
- Developed recommendations to strengthen detection, response, and recovery capabilities.
- Practiced documenting cybersecurity incidents using industry-recognized methodologies.
